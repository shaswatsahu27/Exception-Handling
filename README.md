# 📘 Assignment 6: Exception Handling in Python

## 📌 Problem Statement
This assignment focuses on building simple and safe Python programs that handle:
- Unexpected user inputs  
- Invalid operations  
- Runtime errors  

Using Python’s **exception handling system**.

---

## 🎯 Objectives
- Understand and use `try`, `except`, `else`, and `finally`
- Handle common exceptions like:
  - ValueError  
  - ZeroDivisionError  
  - TypeError  
  - FileNotFoundError  
- Create custom exceptions using `raise`
- Write clean and beginner-friendly code
